# Unit tests for the platform independent logic

Test system is [Unity](https://www.throwtheswitch.org/unity), build with [Ceedling](github.com/ThrowTheSwitch/Ceedling)
Ceedling can be easily installed following their [guide](https://github.com/ThrowTheSwitch/Ceedling/blob/master/docs/CeedlingPacket.md#ceedling-installation--set-up)

## Running the tests

Once Ceedling is installed in local host system, test compilation and execution is straightforward.

	ceedling test:all
